# Painting-Board

<br>

## 소개
HTML, CSS, Vanilla JS를 이용한 미니 쇼핑몰 게임 앱을 만들었습니다.

<br>

## 구현 기능
  - 캔버스에 그림 그리기

<br>

## 문제 사항
  - 로컬 호스트에서 확인 시 PWA 적용사항이 확인 되지만<br>깃헙 페이지 링크로 확인 시 적용 사항이 보이지 않는다.

<br>

## 사용 기능 설명 
  - html의 canvas 테그를 사용했습니다.

<br>

## 개발 환경
  - 개발도구: VSCode, Github
  - 사용언어: HTML, CSS, Vanilla JS
